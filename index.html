BookFinder

Find new books to read!

Submit an author or a book to find new, similar authors or books.

A couple tips to make your searches more effective:

Using an author's full name (first and last) is likely to give better results, as is being precise in spelling and use of special characters.

For example, a search for "Roberto Bolaño" (note the ñ!) will do better than a search for "Roberto Bolano".

If you don't find a review of a specific book, a search for that book's author is more likely to successfully find reviews.

    End of search results!

    Did you find a specific book you're interested in? It might be freely available online at Open Library! To search, enter the ISBN for your book of interest.

    This tool uses the TasteDive API to create the recommendation lists, the New York Times Books API to find NYTimes book reviews, and the Open Library API to search for free online rentals.
